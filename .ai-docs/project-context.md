# TaleX - Project Context 🚀

TaleX là nền tảng phát triển video truyện tranh và hoạt hình ngắn. Hệ thống hỗ trợ 4 nhóm người dùng chính:
1. **Viewer:** Xem video, tương tác thời gian thực, nạp Coin, mở khóa Fast Pass.
2. **Creator:** Quản lý nội dung series, xem Dashboard phân tích doanh thu, nhận chia sẻ doanh thu từ Ads và Premium Subscriptions.
3. **Staff:** Kiểm duyệt nội dung, xử lý vi phạm DMCA và báo cáo.
4. **Admin:** Quản lý kinh tế hệ thống (Coin, Tỷ lệ doanh thu), KYC.
